# FullStack Ecommerce Platform

A modern full-stack ecommerce web application built with the MERN stack, featuring authentication, profile management, product management, and admin functionality.

## Features

### User Features

- User authentication & authorization
- Forgot password & password reset
- Profile image upload
- Username & email update
- Secure account management

### Product Features

- Product CRUD functionality
- Product filtering by:
  - Category
  - Name
  - Size
  - Color
  - Price range

### Admin Features

- Admin-side product management (In Progress)

## Tech Stack

### Frontend

- React
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Router
- React Hook Form
- Zod
- Shadcn UI

### Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Express Validator
- Cloudinary
- MailTrap

## Learning & Improvement

This project was built to strengthen my understanding of full-stack development, authentication systems, state management, scalable architecture, form validation, and frontend-backend integration.

## Installation

```bash
npm install
npm run dev
```
